# AutoDelovi

Platforma za kupovinu auto delova i zakazivanje ugradnje u Srbiji.

## Stack
- Next.js 14 (App Router)
- - Supabase (PostgreSQL + Auth)
  - - Vercel (Hosting)
    - - Tailwind CSS
     
      - ## Setup
      - 1. `npm install`
        2. 2. Kopiraj `.env.example` u `.env.local` i popuni kredencijale
           3. 3. `npm run dev`
